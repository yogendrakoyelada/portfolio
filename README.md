# YK Analytics — Portfolio

This repository contains a production-ready, static portfolio site built with vanilla HTML, CSS and JavaScript.

How to preview:
- Open index.html directly (modern browsers) or serve with a static server (e.g. http-server) or deploy to Netlify.

Replaceable assets:
- assets/resume.pdf — replace with your real resume
- assets/images/* — replace placeholder images

Editing content:
- All page content is driven by JSON files in /data. Edit these to update the site content.
